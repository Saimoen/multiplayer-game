# Installation

Initialiser les dépendances du projet : 

```javascript
/Desktop/multiplayer-game
npm install
cd client
npm install
```

Lancer le projet en local :
```javascript
/Desktop/multiplayer-game
npm start
cd client
npm run dev
```

Le serveur sera lancé à l'adresse localhost:3000 et le jeu se jouera sur l'adresse localhost:5173
